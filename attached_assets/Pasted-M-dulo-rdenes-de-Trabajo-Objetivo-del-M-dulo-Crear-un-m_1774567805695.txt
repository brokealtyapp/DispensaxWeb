Módulo: Órdenes de Trabajo
Objetivo del Módulo
Crear un módulo de Órdenes de Trabajo para gestionar todas las actividades relacionadas con las
máquinas de Dispensax, incluyendo:
●
Gestión de Tickets
●
SLA (tiempos de respuesta)
●
Checklist por tipo de trabajo
●
Asignación de personal (abastecedores / técnicos)
●
Seguimiento y control operativo
Este módulo será el centro operativo para la gestión de fallas, abastecimiento y mantenimiento de
máquinas.
Concepto Principal
Las Órdenes de Trabajo siempre deben estar vinculadas a una máquina.
No se puede crear una orden sin seleccionar primero una máquina.
Cada Orden de Trabajo se generará cuando:
●
Un cliente reporte un problema
●
El sistema genere una alerta
●
Se programe un abastecimiento
●
Se programe mantenimiento preventivo
●
Se requiera instalación
●
Se requiera retiro de máquina
Tipos de Órdenes de Trabajo
El módulo debe incluir los siguientes tipos:
1.
Abastecimiento
2.
Técnico (Correctivo)
3.
Mantenimiento Preventivo
4.
Instalación de Máquina
5.
Retiro de Máquina
Cada tipo de orden debe tener su Checklist específico.
